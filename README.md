# 이더리움 기반 부동산 거래 플랫폼 Dapp
이더리움 블록체인의 화폐인 "Ether" 를 활용하여 부동산 매물 등록 및 거래가 가능한 부동산 거래 플랫폼 입니다.

## 개발 환경
<li>Window 10 OS</li>
<li>Visual Studio Code</li>
<li>MetaMask</li>


## 사용된 기술
### 백엔드
<li>Node.js</li>
<li>Express</li>
<li>IPFS</li>
<li>Pinata</li>
<li>Web3</li>

### 데이터베이스
<li>MongoDB</li>

### 프론트엔드
<li>HTML/CSS/JavaScript</li>
<li>React</li>
